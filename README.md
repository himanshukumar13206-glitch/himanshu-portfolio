## 🚀 **About The Project**

A fully interactive, cyberpunk-themed personal portfolio website built from scratch. It showcases skills, projects, achievements, and blogs with a hacker aesthetic.  
The site includes **Matrix rain**, **particle networks**, **glitch effects**, **flip cards**, and an **admin dashboard** for real‑time content editing.

### ✨ **Key Features**

- 🎨 **Immersive UI** – Cyberpunk design with neon accents, custom cursor, and animated backgrounds.
- ⚡ **Dynamic Content** – All sections (About, Skills, Projects, Blog, Contact) are editable via an admin panel.
- 🔐 **Admin Dashboard** – Login system (`himanshu` / `iam30426avi`) to manage site data without touching code.
- 📝 **Blog with Images** – Add blog posts with optional cover images.
- 📧 **Contact Form** – Sends you an email directly via the user’s mail client.
- 🎵 **Background Music** – Toggleable “River Flows in You” melody using Web Audio API.
- 🖱️ **Fully Responsive** – Looks great on all devices.
- ⌨️ **Keyboard Shortcuts** – `Ctrl+1/2` switch pages, `M` toggles music, `Esc` closes modals.
